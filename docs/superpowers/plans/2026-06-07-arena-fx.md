# Arena FX & Cutscenes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild the Pixel Arena on PixiJS with cinematic cutscenes (boss intro, victory, danger, potion), combo escalation effects, gamified choice/plan display, and a live RPG stats panel; rename the player resource HP → Token.

**Architecture:** Vendored PixiJS v8 (no CDN, no npm for users). `public/arena.js` holds all game code; `public/sequencer.js` is a pure UMD timeline player unit-testable under node. Cutscenes are declarative `{at, do}` data. Hooks/serve stay observer-only; new event kinds `boss_down`, `choice_open`, `choice_made`, `plan_scroll`, `plan_approved` flow through the existing session jsonl → SSE pipe.

**Tech Stack:** Node ≥20 (no new deps), PixiJS 8.6.6 vendored, `node:test`.

**Spec:** `docs/superpowers/specs/2026-06-07-arena-fx-design.md`
**Spec correction:** spec says "new hook script `hook-pretool.js`" — those scripts already exist (cast/resolve mapping). Tasks 6 extends them instead of creating new files.

```js
(function (root, factory) {
  if (typeof module === 'object' && module.exports) module.exports = factory();
  else root.QLSeq = factory();
}(typeof self !== 'undefined' ? self : this, function () {
  'use strict';

  function createTimeline(steps) {
    const sorted = [...steps].sort((a, b) => a.at - b.at);
    return { steps: sorted, i: 0, frame: 0, done: sorted.length === 0 };
  }

  // Returns the steps due this frame, then advances the frame counter.
  function advance(tl) {
    if (tl.done) return [];
    const due = [];
    while (tl.i < tl.steps.length && tl.steps[tl.i].at <= tl.frame) due.push(tl.steps[tl.i++]);
    tl.frame++;
    if (tl.i >= tl.steps.length) tl.done = true;
    return due;
  }

  // Photosensitivity cap: at most maxPerSec flashes, enforced as a minimum
  // frame gap. allow(frame) returns whether a flash may fire now.
  function createGovernor(maxPerSec, fps) {
    const minGap = Math.ceil((fps || 60) / (maxPerSec || 3));
    let last = -Infinity;
    return {
      allow(frame) {
        if (frame - last < minGap) return false;
        last = frame;
        return true;
      },
    };
  }

  return { createTimeline, advance, createGovernor };
}));
```

'use strict';
// Gamified token-cost heuristic. NOT a real estimator — a threat assessment
// for the arena. No LLM calls (observer principle).
function estimateTokens(text) {
  const s = String(text || '');
  const steps = (s.match(/^\s*(?:[-*]|\d+[.)])\s+/gm) || []).length;
  const est = 25000 + steps * 30000 + s.length * 3;
  return Math.max(20000, Math.min(900000, est));
}
function fmtTokens(n) {
  return `≈${Math.round(n / 10000) * 10}k`;
}
module.exports = { estimateTokens, fmtTokens };
