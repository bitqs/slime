# Questline i18n Gap-Closure Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the remaining i18n gaps — localize the three CLI command scripts, add a zh/en catalog parity test, add a language step to setup, and ship a Chinese README.

**Architecture:** The locale core already exists (`scripts/lib/locale.js`: prompt-language tally with `config.json` `lang` override; catalogs at `data/locales/{en,zh}.json`; `t(key, lang)` + `fmt(tpl, vars)`). This plan only threads it through the three command scripts and adds the missing assets. English output must stay byte-identical when locale resolves to `en`.

**Tech Stack:** Node built-ins only (zero deps), `node:test`, catalogs as JSON.

**Concurrency note:** A parallel session works in this tree — see `COORDINATION.md`. Stick to claimed files; `data/locales/*.json` is shared append-only; `git add` explicit paths only.

**Spec note:** The approved spec (`docs/superpowers/specs/2026-06-07-i18n-design.md`) was partially superseded by the as-built P3 implementation (prompt-language tally instead of `LANG` env; `data/locales/*.json` instead of `scripts/locales/*.js`; boss names baked per-language at creation; tips in `data/tips.zh.json`). Task 5 records this addendum in the spec.

- [ ] **Step 3: Rewrite `scripts/defeat.js`**

```js
#!/usr/bin/env node
const state = require('./lib/state');
const boss = require('./lib/boss');
const { t, fmt } = require('./lib/locale');
const fs = require('node:fs');

const cwd = process.argv[2] || process.cwd();
try {
  if (!fs.existsSync(boss.bossPath(cwd))) {
    console.log(t('cmd.noBoss'));
    process.exit(0);
  }
  const b = boss.loadOrCreate(cwd, '');
  const prof = state.readProfile();
  prof.milestones.push({
    boss: b.name, date: new Date().toISOString().slice(0, 10),
    turns: b.turns || 0, project: cwd,
  });
  state.writeProfile(prof);
  boss.clear(cwd);
  console.log([
    fmt(t('cmd.defeated'), { name: b.name }),
    fmt(t('cmd.recorded'), { n: prof.milestones.length }),
    t('cmd.defeatSage'),
  ].join('\n'));
} catch (e) {
  console.log(fmt(t('cmd.glanced'), { msg: e.message }));
}
process.exit(0);
```

#!/usr/bin/env node
const state = require('./lib/state');
const { t, fmt } = require('./lib/locale');
try {
  const prof = state.readProfile();
  const lines = [t('cmd.wall'), ''];
  if (!prof.milestones.length) lines.push(t('cmd.wallEmpty'));
  for (const m of prof.milestones) {
    lines.push(fmt(t('cmd.wallEntry'), { date: m.date, boss: m.boss, turns: m.turns, project: m.project }));
  }
  lines.push('', fmt(t('cmd.career'), { turns: prof.totals.turns, dmg: prof.totals.dmg, kills: prof.totals.kills }));
  console.log(lines.join('\n'));
} catch (e) { console.log(fmt(t('cmd.wallUnreadable'), { msg: e.message })); }
process.exit(0);
